# TMP-Spring-20-Hackathon
Electron app to aid in running blt project

 - init: yarn install && yarn build
     command: npm start & node app.js

```bash
# start a livereaload server 
npm start 

# start the webserver
node app.js
```


```bash
# run the electron app
npm run electron

# run the web app
npm run web
```
